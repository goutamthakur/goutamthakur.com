# Portfolio website code
If you like the theme you are free to use it. Or for any reason want to help me pull request are always welcomed 

### Project setup 

1. Clone the repository 

`
git clone https://github.com/goutamthakur/goutamthakur.com.git
`

2. Install the dependencies

`
npm install 
`

3. Start the project 

`
npm run dev
`
